# Digital Clapperboard

ブラウザで動作するシンプルで高機能なデジタルカチンコ（スレート）アプリケーションです。
映像制作の現場で、シーン、カット、テイクの管理やタイムコードの同期に使用できます。

## 特徴

*   **基本情報の入力**: Production名、Roll、FPS（フレームレート）、Director、Camera名などを編集可能。
*   **カウンター機能**: Scene、Cut、Takeの数値をボタンで簡単に増減。リセット機能付き。
*   **タイムコード**: 設定したFPSに基づいた正確なタイムコード計測。
*   **カチンコ機能**: "CLAP"ボタンでビープ音と画面フラッシュを実行し、映像と音声の同期をサポート。
*   **テーマ切り替え**: 視認性に合わせてダークモード（BLACK）とライトモード（WHITE）を切り替え可能。
*   **レスポンシブデザイン**: PC、タブレット、スマートフォンなど様々な画面サイズに対応。

## 使い方

1.  以下のファイルをすべて同じフォルダに配置します。
    *   `index.html`
    *   `style.css`
    *   `script.js`
2.  `index.html` をモダンブラウザ（Chrome, Safari, Edgeなど）で開きます。
3.  画面上の各項目をクリックして編集し、撮影を開始します。

## ファイル構成

*   `index.html`: アプリケーションの構造とレイアウト。
*   `style.css`: デジタルフォントの読み込みやボタンのデザインなどのスタイル定義。
*   `script.js`: タイマー計測、カウンター操作、音声再生などのロジック。

## 使用技術

*   HTML5
*   CSS3 (Tailwind CSS - CDN経由で使用)
*   JavaScript (Vanilla JS)

## 注意事項

*   Tailwind CSSとデジタルフォントをCDN経由で読み込んでいるため、初回読み込み時にはインターネット接続が必要です。
*   音声再生のため、ブラウザの音声再生許可が必要な場合があります。
